Noethrion η — verify it yourself

Don't trust us. Re-derive every leaf, replay the Merkle proof, and check it against the on-chain root — entirely in your browser. No install, no server.

The verifier is a self-contained WebAssembly build of the same Go code that powers the command-line noethrion-verify node — keccak256 leaf re-derivation, sorted-pair Merkle replay (OpenZeppelin semantics), and the on-chain batches(uint64) read. Nothing is sent anywhere except your chosen RPC endpoint.

Source: node/go/wasm/ in the public repo.
